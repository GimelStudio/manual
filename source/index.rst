Gimel Studio Manual
===================

Welcome to the manual for Gimel Studio, the free and open-source, **cross-platform, non-destructive, node-based 2D image editor** focused on simplicity, speed, elegance and usability.

.. note::

    Gimel Studio is currently in early stages of development and is not stable. Not all features are implemented and things may be broken at any time.

.. note::

   This manual is still being written. If you would like to help in the documentation effort, please let us know.
